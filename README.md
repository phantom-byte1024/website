# My Personal Website

This is a basic template for a personal website built with HTML, CSS, and JavaScript.

## Files

- `index.html`: The main HTML structure of the website.
- `style.css`: Basic styling for the website.
- `script.js`: JavaScript for interactivity, including smooth scrolling.

## Usage

1. Open `index.html` in a web browser to view the website.
2. Customize the content in `index.html` to reflect your personal information.
3. Modify `style.css` to change the appearance.
4. Add more functionality in `script.js` as needed.

## Customization

- Replace [Your Name] and other placeholders with your actual information.
- Add more sections or projects as desired.
- Update the email address in the contact section.

## Deployment

You can deploy this static website to any web hosting service that supports HTML, CSS, and JS files.
